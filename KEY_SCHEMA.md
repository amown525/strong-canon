# Strong RU Canon — KEY_SCHEMA

Это репозиторий “живых ключей” (canon_ru / human_ru) поверх Strong’s IDs.
Цель: превратить номера Strong в короткие терапевтические “формулы внимания”.

---

## 0) Термины

- **key** — запись Strong: `H####` или `G####`.
- **canon_ru** — “канон” (3–7 слов, максимально прямой).
- **human_ru** — “человеческие отклики” (3–7 глаголов/образов, через ` · `).
- **prompt_ru** — один вопрос, который запускает самонаблюдение.
- **refs** — ссылки/употребления в Писании (минимально: книга/глава/стих).

---

## 1) Канонический формат одной записи (объект)

### Минимум (MVP)
```json
{
  "id": "H7225",
  "canon_ru": "начало · первое · главное",
  "human_ru": "начать · решить · взять главное",
  "prompt_ru": "❓ Что для тебя сейчас «первое»?"
}
