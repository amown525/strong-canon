# strong-canon
A living canon of human meaning based on Strong’s Concordance.

# STRONG · RU

Публичный канон живых ключей —
язык работы с собой через слово, тело, решение и путь.

Это не психология.
Не религия.
Не коучинг.

Это операционная система смыслов,
собранная из первичных слов и человеческих откликов.

## Категории
- Семья
- Деньги
- Страх / Боль
- Власть / Решение
- Тело
- Путь

## Принцип
Каждый ключ — это:
- канон
- живые глаголы
- вопрос-активация
